installation

- open the brainbugz.zip archive using your favourite zip-tool
- extract the files from the folders to the specified destinations:

folder contents destination
plugin brainbugz.mll ~/maya/x.x/bin/plug-ins
mel bbCreateUI.mel
bbDeleteUI.mel
AEbbSteeringDesireTemplate.mel
AEbbCombineForces.mel
AEbbConnectionEditor.mel

~/maya/x.x/scripts

(make sure it's in your MAYA_SCRIPT_PATH)

images

bbCEA1.xpm
bbCEA2.xpm
bbCEA3.xpm
out_bbCombineDesires.xpm
out_bbSteeringDesire.xpm

 ~/maya/x.x/icons
fti-icons bbCombineDesires.fti
bbSteeringDesire.fti
 ~/AW/Mayax.x/icons

- start Maya
- load plugin into Maya, using the Plug-in Manager (Windows > Settings/Preferences > Plug-in Manager...)
- a new menu "brainbugz" will appear in the menu-bar, use this to envoke all commands related to brainbugz

IMPORTANT:
brainbugz uses the following node id's 0x3a001, 0x3a002 - these lie in the range designated by A|W for private use. If you are expriencing problems using the nodes, make sure these id's are not already used by other plugins!
