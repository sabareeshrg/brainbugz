 known issues and limitations

- not complete nurbsSurfaces area is used for target destribution
- multiple knot targets with same position

If you find a bug, please report to carsten@kolve.com

 future enhancements

all these depend on time, feedback and motivation :)

- new steering desires: formation, containment
- new combine mode: priorized dithering
- node to constrain bug motion to different physical parameters (surface hugging, flight etc.)
- performance: eliminate unnessecary computations (you'd be shocked if you knew how much there are!), spatial data structure to speed up query of nearby bugs, node to store reusable computational data to speed up calculations
- event / location based steering desire changes (already possible to a certain degree, extend that degree)
- inner states for bugs (fear, hunger, anger etc.)
- integration of non-linear trax animation with beavioural animation engine
- kill all bugs, kill 'em all!
- faster, better, harder, stronger

Suggestions, ideas? > carsten@kolve.com
